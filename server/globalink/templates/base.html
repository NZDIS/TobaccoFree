About

To collect data from multiple sites, globally, many times a day, over and over again, would be expensive. So expensive, that study like that is impossible to run through traditional methods.

The idea is to ask volunteers to help. YOU will help to conduct the study and do the research.

  • Anyone can participate
  • Register
  • Download and install NZDIS TobaccoFree Android application
  • Agree to University of Otago »Ethical Conduct and Honour Code«
  • Go outside, stand by a busy street, and start logging the data

The application allows easy logging of occurances of smoking in cars. Install it, go out, and collect data about particular location. Instructions are provided within the application.
